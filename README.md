# ML Learning Platform

Interactive Machine Learning learning platform with real-time model simulations, hands-on coding exercises, and gamified education.

## Features
- Parameter Playground - Adjust ML parameters in real-time
- Live Data Lab - Write and execute Pandas code
- Algorithm Visualizer - Step-by-step algorithm animations
- Gamification - XP, levels, streaks, leaderboards

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express, MongoDB
